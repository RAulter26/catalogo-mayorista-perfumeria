Pack de investigación — Perfumería Mayorista

Este paquete se armó como base para entregar a Claude y montar una demo/componente de catálogo mayorista de perfumería.

Qué trae:
- data/productos_60_perfumeria_mayorista.csv
- data/productos_60_perfumeria_mayorista.json
- data/resumen_categorias.md
- images_placeholders/ con 60 tarjetas visuales placeholder originales
- PROMPT_PARA_CLAUDE.md

Importante:
- Los 60 productos se recopilaron a partir de páginas públicas de Falabella, Medipiel, Disfragancias y Victoria's Secret Beauty Colombia.
- Las imágenes de productos de terceros NO se incluyeron dentro del ZIP.
- En su lugar, este ZIP trae placeholders propios para maquetación rápida.
- La columna source_url deja rastreable de dónde salió cada referencia.
- Algunas clasificaciones como audience y suggested_category son propuestas internas para organizar la demo comercial.
